# Contributing

感谢你对 `LiveAssassin` 的贡献。

## 提交方式
- 使用 Pull Request 提交改动。
- 提交前请确保项目可编译。
- 新功能请附带简要说明和使用方式。

## 代码规范
- Java/Kotlin 代码保持简洁，避免无意义复杂化。
- 尽量避免提交与本需求无关的大范围格式化改动。
- 提交前清理调试日志与临时代码。

## Issue
- 提交问题时请提供：
  - 设备型号/Android 版本
  - 采集卡型号（VID/PID）
  - 关键日志（`UVCCamera` / `libusb` / `AudioTrack`）
